Region Mod by Zmiize94[edit]

Back up the game's original executable before using this mod.
Allows the option to play between the North America, Japanese, and a fixed European version of this collection, as well as improving the sound quality of the Sega Genesis/Mega Drive emulation in this collection. Details on how to use this mod's features can also be found in this video and the included readme file.
Installing Region Mod[3]
  1. Download the mod here and extract the contents of the zipped archive to <path-to-game>.
  2. Replace the default executable with the appropriate ones from <path-to-game>Modded EXE (U for North America, J for Japan, and E for Europe) and rename it to MegaCollectionPlus.

Unlock all games instantly[edit]

Adding the following to the game's configuration file[6]
  1. Go to the configuration file location.
  2. Open SETTINGS.INI.
  3. Scroll down to the section under [ETC] and add the following underneath the line DELETE_TEX_LINE: SELECT_ALL_GAMES=1.
  4. Save the changes. You should now be able to play all games featured in this collection.
